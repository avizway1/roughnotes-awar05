A Record : Address Record : Maps domain name to IPv4
AAAA Record : Address Record : Maps domain name to IPv6
CNAME Record : Cananical Name : It maps to a domain to another domain. (fb.com --> facebook.com)
MX Record : Mail Server/Exchange Records for email handlings..
TXT Record : Stores text data.. used to domain verification / other purposes..
PTR Record : Reverse DNS lookup..

Alias Record : AWS Specific Record, it helps to map resources i.e; ELB, Cloudfront, S3..
